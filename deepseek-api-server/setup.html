How to get Google ID token:
1. Click the button below to sign in with Google
2. Complete the Google authentication flow
3. The ID token will be automatically extracted and used
4. Alternatively, you can manually get the token from browser DevTools
How to get your token:
1. Open DeepSeek in your browser
2. Login to your account
3. Open Developer Tools (F12)
4. Go to Application → Local Storage
5. Copy the 'access_token' value
